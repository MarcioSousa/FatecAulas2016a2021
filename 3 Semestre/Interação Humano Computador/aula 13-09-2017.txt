Principais tags/atributos de acessibilidade

*alt  -> texto alternativo da imagem
*<noscript> ->texto exibido qundo o script naõ é carregado
*summary -> Resumo da tabela
*abbr -> Abreviação de coluna
*thead, tbody e tfood -> Utilizado para separar as diferentes partes da tabela

=====Principais tags/atributos da web semântica=====
<nav> -> Área de navegação
<article> -> Define um artigo, notícia ou produto
<section> -> Define a seção
<header> -> Cabeçalho da página
<footer> -> Rodapé da página
<aside> -> Área lateral do site
