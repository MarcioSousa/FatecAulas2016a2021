              WEB SEMÂNTICA

Permite que não apenas os usuários possam entender as páginas web,
mas principalmente as máquinas.

Principais tags da web Semântica (disponíveis a partir do HTML5)
<nav> Define a área de navegação (menu) do site.
<section> Divide o site em várias seções (ou áreas) específicas.
<header> Área de cabeçalho do site
<folder> Área de rodapé do site
<article> Define a área de conteúdo do site (Pode ser uma notícia, um produto, etc)
<aside> Utilizado para definir uma área "à parte" do site. Ex: propagandas ou anúncios.
